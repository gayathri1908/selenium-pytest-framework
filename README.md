# Selenium Pytest Framework

## Overview
This project is a **Selenium-based automated testing framework** using **Python** and **Pytest**. It is designed to test web applications, specifically login functionality, with support for:

- Parametrized tests
- Automatic HTML reports
- Screenshot capture on test failures
- Logging of test execution

The framework helps automate repetitive testing tasks, track failures with screenshots, and generate readable test reports.

## Features
1. **Parametrized Tests:** Easily run the same test with multiple sets of data.
2. **HTML Reports:** Generates detailed HTML reports for each test run.
3. **Screenshots on Failure:** Automatically captures screenshots when a test fails.
4. **Logging:** Logs important information such as usernames, passwords, and test steps.
